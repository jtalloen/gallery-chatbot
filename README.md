# Gallery263 Chatbot

A simple Gemini powered chatbot for Gallery263. The live version is [[here](https://gallery263-chatbot.streamlit.app/)].

<br>

| File | Description |
|:-----|:------------|
| `GalleryChatbot.py` | Main application code with Streamlit chatbot with Gemini AI integration, custom styling, and HuggingFace fallback |
| `requirements.txt` | Python dependencies for the project |
| `logo263.png` | Gallery263 logo used in the app header and favicon |
| `.streamlit/config.toml` | Streamlit configuration settings |
